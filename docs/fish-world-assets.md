# Fish World Asset Checklist

## Minimum Prototype Assets

- player fish placeholder GLB
- 2 prey fish GLBs
- 2 predator fish GLBs
- reef rock set
- kelp set
- corpse / meat pickup mesh
- bubble texture
- caustic texture
- basic underwater ambience

## Production Asset Categories

### Characters
- player fish evolution forms
- prey species set
- neutral species set
- predator species set
- apex species set

### Environment
- shallow reef region kit
- kelp forest region kit
- open blue region props
- rocky depth kit
- trench / abyss modules
- natural boundary set

### FX
- bubbles
- water particulate
- impact bite FX
- blood / damage FX if desired
- bioluminescent glow FX

### UI
- radar icon pack
- growth / evolution icons
- region markers
- objective markers

## Recommended Format

- models: `glb`
- textures: compressed browser-friendly textures when finalizing
- audio: `.wav` for source, optimized runtime exports later
